# BED2-H26 Læringsrom – publisert nettsted

Dette offentlige depotet inneholder kun den ferdigbygde, statiske utgaven av læringsrommet. Kildeprosjektet, transkripsjonene og analysegrunnlaget ligger i et separat privat GitHub-prosjekt.

Nettstedet bruker Supabase til innlogging og synkronisering av personlig studieprogresjon. Ingen hemmelig servernøkkel ligger i denne leveransen.
